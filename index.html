Projects

Academic


Computer Graphics

This class provided a creative outlet for myself to explore and I really enjoyed the animation unit. In particular I am very proud of my "Endless Runner" robot simulation that involved static and dynamic obstacles, a skybox, textures, and a falling leaves particle effect. Also included are a model manipulation program, a realistic bubble blowing simulation, and the previous version of the animated robot involving two different projections and views. These were made using OpenGL 2.1 and Java 8.

Software Engineering II

The "partEz" application is what my group created for our Software Engineering II class using a Laravel, MySQL, and PHP stack. The app exists on both web and android platforms. The design aim is for the app to be used to plan, organize, share, and otherwise document parties. In additon to contributing to this code base via implementation of the Invitations page, Event View, and testing, I also designed and created the logo. The codebase is linked here on my GitHub.


Personal


Baseball Board Game

In the 1970's, my great-grandfather decided to make his own board game about major league baseball. Having just completed the computer graphics class in my university, I plan to recreate his game in Unity with the tools I have learned so that it may be preserved electronically in my family. It will also give me a chance to learn Unity for the first time while gaining practice developing my own games, which is something I've aspired to do since I was very young.




Grade Point Awesome

As an exploratory learning activity, myself and another classmate are making an iOS app that can track grades on various provided grade scales. This was inspired by having to manually enter Canadian GPA scales into current store apps, as well as predicting what exam grades are needed to achieve a certain letter grade in a class. This is an ongoing project during our university terms and we are taking the opportunity to learn Xcode and iOS using academic resources.
