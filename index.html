Projects

partEz

This is a project in progress for my Software Engineering II class. The partEz app will be used to plan, organize, share, and otherwise document parties, socials, get-togethers, gatherings, and other various events. Users will be able to fully plan a party or event and will be able to keep attendees updated on announcements and changes in plans.

Major League Baseball

This is my own personal project. In 1991, my great-grandfather decided to make his own board game about major league baseball. My task is to recreate this game in Unity so that it may be preserved as such, while gaining practice developing my own game!
